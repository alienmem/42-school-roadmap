## üí° Critical Concepts

### Pointers (C01)
```c
int x = 42;
int *ptr = &x;    // ptr holds address of x
*ptr = 100;       // x is now 100
```

### Dynamic Memory (C07)
```c
int *arr = malloc(5 * sizeof(int));  // Allocate
if (!arr) return (NULL);             // Check!
// ... use arr ...
free(arr);                            // Free!
arr = NULL;                           // Nullify
```

### Linked Lists (C12)
```c
typedef struct s_list {
    void          *content;
    struct s_list *next;
} t_list;
```

### Binary Trees (C13)
```c
typedef struct s_btree {
    struct s_btree *left;
    struct s_btree *right;
    void           *item;
} t_btree;
```

## üö´ Common Pitfalls

‚ùå **Not checking malloc return**
```c
int *arr = malloc(10 * sizeof(int));
arr[0] = 42;  // SEGFAULT if malloc failed!
```

‚úÖ **Always check!**
```c
int *arr = malloc(10 * sizeof(int));
if (!arr)
    return (NULL);
arr[0] = 42;  // Safe
```

‚ùå **Forgetting to free**
```c
char *str = malloc(100);
return (str);  // Memory leak!
```

‚ùå **Using after free**
```c
free(ptr);
*ptr = 42;  // Undefined behavior!
```

‚ùå **Off-by-one errors**
```c
char str[10];
str[10] = '\0';  // Buffer overflow!
```
