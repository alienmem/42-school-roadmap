## üìñ What is the Piscine?

The **Piscine** (French for "swimming pool") is 42 School's selection bootcamp. Traditionally, it's 26 days of intensive C programming where students "sink or swim."

As a self-study student, I'm following the same curriculum at my own pace, completing all exercises with the same rigor and standards as campus students.

## üéì Learning Philosophy

**The 42 Way**:
1. **Read the subject** carefully (2-3 times)
2. **Plan on paper** before coding
3. **Code from scratch** (no copy-paste!)
4. **Test obsessively** (edge cases, NULL, limits)
5. **Zero tolerance** (memory leaks, norminette errors)
6. **Ask for concepts** (never solutions)

**The struggle IS the learning.** Every segfault teaches something.
