# Projects 💡

This folder is intended to contain **your implementations** of the 42 School projects.

Organize your repositories or source code here, keeping it parallel to the corresponding specifications found in the `subjects/` folder.

Example layout:

projects/
├── libft/
├── ft_printf/
├── get_next_line/
└── push_swap/


Each project folder should ideally contain:
- The full source code
- A `Makefile`
- A `README.md` describing compilation, usage, and testing instructions
