# 🌟 42 School Portfolio

Welcome to my **42 School Curriculum Portfolio** — a complete collection of projects, exercises, and official materials from the **42 School** program.  
This repository is designed both as a personal archive and as a resource hub for fellow students navigating the same journey. 🚀

---

## 🏫 About 42

**42** is a tuition-free, peer-to-peer coding school that focuses on **project-based learning** and **collaboration** rather than traditional lectures.  
The curriculum covers everything from low-level programming in C to high-level topics like DevOps, web development, graphics, and AI.

## 📘 Folder Summaries

### 🏊 **`piscine/`**
Contains all materials from the **42 Piscine**, including C and shell exercises, rush projects, and exam preparations.

### 🎓 **`subjects/`**
Holds all official **project subjects** (PDFs, French and English) and related resources.  
Acts as the reference library for the entire Common Core and CPP cursus.

### 💻 **`projects/`**
Stores **my implementations** of the 42 projects — organized by level, matching the same structure as `subjects/`.

### 🧾 **`docs/`**
Includes the **Norm v4**, technical notes, and other helpful files used during studies.
